# MaxParallelTasks

Parameter `MaxParallelTasks` defines the maximum number of parallel tasks for the DCV runtime.

## Example

```json
{
    "MaxParallelTasks":4
}
```

## Parameter Summary

| MaxParallelTasks Parameter Summary |
| :------------- |
| **Type**<br>*int* |
| **Range**<br>[1, 256] |
| **Default Value**<br>4 |
