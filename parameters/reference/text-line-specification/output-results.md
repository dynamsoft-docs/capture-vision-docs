# OutputResults

Parameter `OutputResults` defines whether to enable the output of the `TextLineSpecification` object.

## Example

```json
{
    "Name": "tls_0",
    "OutputResults" : 0
}
```

## Parameter Summary

| Parameter Details |
| :----------------------------------- |
| **Type**<br>*int* |
| **Range**<br>0,1 |
| **Default Value**<br>0. It does not output the text line result for the `TextLineSpecification` object. |
