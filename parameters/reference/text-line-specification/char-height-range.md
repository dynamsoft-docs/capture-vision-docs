# CharHeightRange

`CharHeightRange` defines the range of the character height.

```json
{
    "CharHeightRange": [ 800, 1000, 1 ]
}
{
    "CharHeightRange": {
        "Minimum": 800,
        "Maximum": 1000,
        "MeasureInThousandth": 1
    }
}
```

| **Type**<br>*int Array* |
| **Range**<br> |
