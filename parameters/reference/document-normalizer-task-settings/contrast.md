# Contrast

Parameter `Contrast` specifies the contrast of the normalized image.

## Example

```json
{
    "Contrast": 0
}
```

## Parameter Summary

| Contrast Parameter Summary |
| :------------------------ |
| **Type**<br>*int* |
| **Range**<br>[-100,100] |
| **Default Value**<br>0 |
