# Brightness

Parameter `Brightness` defines the brightness of the normalized image.

## Example

```json
{
    "Brightness": 0
}
```

## Parameter Summary

| Brightness Parameter Summary |
| :------------- |
| **Type**<br>*int* |
| **Range**<br>[-100, 100] |
| **Default Value**<br>0 |
