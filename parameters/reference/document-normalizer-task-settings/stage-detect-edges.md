# Detect Edges Stage

The `Detect Edges Stage` is designed at the stage level to detect edges of quadrilaterals that meet the specified conditions.

```json
{
    "Stage": "SST_DETECT_EDGES"
}
```

## Stage

The stage is named `SST_DETECT_EDGES`.
