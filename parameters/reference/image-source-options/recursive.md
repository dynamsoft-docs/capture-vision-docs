# Recursive

Parameter `Recursive` indicates whether to fetch files recursively.

## Example

```json
{
    "Recursive": 1
}
```

## Parameter Summary

| Recursive Parameter Summary |
| :-------------------------- |
| **Description**<br>Use 0 (false) and 1 (true) to define whether to fetch files. |
| **Type**<br>*int* |
| **Value range**<br>[0,1] |
| **Default Value**<br>1 |
