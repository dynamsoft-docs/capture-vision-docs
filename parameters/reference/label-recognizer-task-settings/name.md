# Name

Parameter `Name` represents the name of a `LabelRecognizerTaskSetting` object, which serves as its unique identifier.

## Example

```json
{
    "Name" : "lr_0"
}
```

## Parameter Summary

| Parameter Details |
| :----------------------------------- |
| **Type**<br>*String* |
| **Default Value**<br>It must be a mandatory setting value. |
| **Remarks**<br>It must be a unique name. |
