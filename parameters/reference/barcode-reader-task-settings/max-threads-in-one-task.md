# MaxThreadsInOneTask

Parameter `MaxThreadsInOneTask` defines the maximum threads that can be consumed in one task.

## Example

```json
{
    "MaxThreadsInOneTask":4
}
```

## Parameter Summary

| MaxThreadsInOneTask Parameter Summary |
| :------------- |
| **Type**<br>*int* |
| **Range**<br>[0, 4] |
| **Default Value**<br>4 |
