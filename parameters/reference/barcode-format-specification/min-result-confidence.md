# MinResultConfidence

`MinResultConfidence` defines the minimum confidence of the result.

## Example

```json
{
    "MinResultConfidence": 0
}
```

## Parameter Summary

The structure of the `MinResultConfidence` is shown as follow:

| MinResultConfidence  Parameter Summary |
| :--------------------------------- |
| **Type**<br>*int* |
| **Range**<br>[0, 100] |
| **Default Value**<br> 30|
